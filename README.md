# Spam-Email-Detection-CEAS_08-dataset-
This repository contains the entire models used for spam email detection.

## All models used
Traditional ML Models 
- Naive Bayes
- Random Forest
- SVM via SGD Classifier
- Logistic Regression

Deep Learning Model
- LSTM

Transformer Model
- BERT mini

*PS: This is built for Google Colab. Please change the file directory to your own if you want to use it using other environment.
